Resetting Passwords

The instructions in this topic explain how to reset a user's password. When following these instructions, ensure you are logged in to the BounceTogether platform as an administrative user or one with Teacher permissions.

  1. From the top menu, click Enrolment / Reset User's Password
  2. From the Class or Group combo box, click the group that the student belongs to
  3. In the Person combo box, click the name of the pupil who's password you want to reset
  4. A prompt will display confirming the password has been reset to "password"

That's it! The user will now be able to log in with password. We strongly recommend this password is changed when the user next logs in.
