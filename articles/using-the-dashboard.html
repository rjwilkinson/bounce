Viewing Results on the Dashboard

Find out how you can use the dashboard to view survey results and run comparisons

Navigation
Posted 16/09/2018 @Bob Wilkinson

Introduction

BounceTogether has been designed to make it easy for you to view and compare the results of completed surveys. This article demonstrates how you can navigate the platform to view the results of a survey you have issued to a group of users. If you want to have even more oversight of the results and run comparisions between groups/classes, check out the Dashboard.
